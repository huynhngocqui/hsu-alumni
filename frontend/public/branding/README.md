Drop branding images into these folders using the exact file names below.

Navbar images:
- /branding/navbar/logo-default.png: desktop logo for the default white navbar state.
- /branding/navbar/logo-scrolled.png: compact logo for the sticky blue navbar state after scroll. Prefer a light or white version.

Footer images:
- /branding/footer/footer-brand.png: footer brand image shown in the left brand block.
- /branding/footer/footer-background.jpg: optional footer background image or texture. The UI adds a dark overlay automatically.

Home images:
- /branding/home/home-hero-banner.jpg: wide banner image for the top image section on the homepage.

Recommended:
- Use transparent PNG for logo files.
- Keep the scrolled logo optimized for dark blue backgrounds.
- If you want to use SVG or different file names, update the paths in src/config/site.js.
